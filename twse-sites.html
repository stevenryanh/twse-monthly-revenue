TWSE 每月營收 — 開發站台

本機開發入口：服務連結、API 測試、常用指令。用 file:// 直接開即可。

服務連結

  • 前端 Web（Vite dev server）http://localhost:5173 Vue 3
  • API（範例：查 2330）http://localhost:5080/api/revenues/2330 .NET 6
  • Swagger UIhttp://localhost:5080/swagger 僅 Development
⚠️ Swagger 與 CORS 只在 Development 環境開啟。./dev.sh 以 --no-launch-profile 啟動（落在 Production），所以 /swagger 會 404、前端跨來源也未放行。要啟用兩者，改用下方「開發指令」裡的 Development 模式啟動 API。

API 測試入口

  • GET 查詢某公司各月營收（最新月在前） curl http://localhost:5080/api/revenues/2330
  • POST 新增一筆營收（依主鍵 upsert） curl -X POST http://localhost:5080/api/revenues \ -H "Content-Type: application/json" \ -d '{ "companyCode": "2330", "dataYearMonth": 202405, "reportDate": 20240610, "companyName": "台積電", "industry": "半導體", "currentMonthRevenue": 176540000, "lastMonthRevenue": 159000000, "lastYearMonthRevenue": 132000000, "moMPercent": 11.03, "yoYPercent": 33.74, "cumCurrentRevenue": 800000000, "cumLastYearRevenue": 600000000, "cumDiffPercent": 33.33, "remark": "測試資料" }'

開發指令

  • ▶
    啟動全部（DB + API + Web，先殺舊 process）./dev.sh
  • ↺
    重啟全部./dev.sh restart
  • ■
    停止 API / Web（MSSQL 容器保留）./dev.sh stop
  • 📋
    即時 log（API + Web + init-db）tail -f logs/*.log
  • 🧪
    Development 模式啟動 API（開 Swagger + CORS）ASPNETCORE_ENVIRONMENT=Development ~/.dotnet/dotnet run \ --project api/src/TwseRevenue.Api
  • ✅
    執行所有測試（net6 在 ~/.dotnet）~/.dotnet/dotnet test api/TwseRevenue.sln
  • 🗄️
    重建資料庫（建表 + 預存程序）bash scripts/init-db.sh

資料庫（MSSQL @ 1433）

  • DB 查最新 10 筆營收（密碼從 .env 讀，不寫死）source .env && docker exec twse-mssql /opt/mssql-tools18/bin/sqlcmd \ -S localhost -U sa -P "$MSSQL_SA_PASSWORD" -C -d TwseRevenue \ -Q "SELECT TOP 10 * FROM dbo.MonthlyRevenue ORDER BY DataYearMonth DESC"
  • ⌨ 進互動式 sqlcmd（密碼從 .env 讀）source .env && docker exec -it twse-mssql /opt/mssql-tools18/bin/sqlcmd \ -S localhost -U sa -P "$MSSQL_SA_PASSWORD" -C -d TwseRevenue
